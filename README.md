## Notes

1. put assets in `public` folder.
1. access public folder in the `src` code using `%PUBLIC_URL%`

## Coding process

1. `elm-app start` to start dev server
1. edit `src` code
1. `elm-app build` for production
1. `gh-pages -d build` to deploy to github.
